# 🚀 Space Word Quest

**Space Word Quest** is a space-themed word guessing game where players sharpen their vocabulary by completing cosmic terms under pressure. With progressive difficulty levels, point multipliers, and alien reactions, it’s both fun and educational!

## 🌌 Features

- 🔠 Word puzzles with missing letters
- 🌍 Three difficulty levels:
  - **Easy:** Short words, 1 missing letter
  - **Medium:** Medium-length words, 1–2 missing letters
  - **Hard:** Long words, 2–3 missing letters
- 🧠 Vocabulary hints (with point penalty)
- 🔊 Sound effects (toggleable)
- 👽 Animated alien alerts
- 🏆 Achievements and combo multipliers
- 📊 Game stats: best score, games played, streaks
- 🎨 Starfield animated background

## 🕹 How to Play

1. Launch the game and click **Start Game**
2. Choose your desired difficulty level
3. Follow the instructions shown
4. Guess the missing letters of each word in order
5. Use hints if needed (but lose points!)
6. Keep your streak alive for bonus multipliers
7. Try to complete all words before time runs out or lives run out!

## 🛠 Installation

1. Clone or download this repository.
2. Make sure you have all three files in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in your browser to start playing.
